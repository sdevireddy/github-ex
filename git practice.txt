1) Create a branch
git checkout -b feature/test

2)Add or Modify files 

3) Revert uncommitted files 
git checkout -- file_name
git checkout -- . (revert all files in repo)

4) Add or Modify files
5) commit changes
git commit -am/m "comment)
git push origin feature/test
6) Revert committed changes
git revert (provide hash number)
7) Remove few commits from repo / roll back last commit
git reset -- hard feature/test
git push --force origin feature/test

8) Shoe details of one commit 
git show commit number
9) git last commit log
git log -1
